## Pepsi Website 
A simple, interactive two-page landing page built for a Pepsi showcase project.

Features
Dynamic Theme: Clicking on different cans changes the background color and text color smoothly while keeping the main product highlighted.

Product Showcase: A clean gallery page that displays all products side-by-side with hover animations.

Fully Responsive: Designed to look good on desktops, tablets, and smartphones.

Technologies Used
HTML5

CSS3 (Flexbox & Keyframe Animations)

JavaScript (DOM Manipulation)
